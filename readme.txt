Страницы:
[] Home (Главная)
[] Open Menu + hover button (Открытое меню + кнопка при наведении)
[] Workouts (Тренировки)
[] Exercise (Упражнение)
[] Profile (Профиль)
[] Create workout (Создание тренировки)
[] Create exercise (Создание упражнения)

Компоненты:
    [] Home (Главная):
       [] Изображение пользователя.
       [] Кнопка "New".
       [] Название упражнения.
       [] Время выполнения упражнения.
       [] Уровень сложности упражнения.
       [] Прогресс выполнения упражнения.
    [] Open Menu + hover button (Открытое меню + кнопка при наведении):
        [] Кнопка для открытия меню.
        [] Выпадающее меню с пунктами:
            [] Workout (Тренировка)
            [] Create new (Создать новое)
            [] Profile (Профиль)
            [] Logout (Выйти)
    [] Workouts (Тренировки):
        [] Список упражнений с иконками:
            [] Push-ups (Отжимания)
            [] Press of shoulders (Жим для плеч)
            [] Biceps curl (Сгибание бицепсов)
            [] Leg calf (Икры ног)
            [] Jump rope (Скакалка)
    [] Exercise (Упражнение):
        [] Изображение оборудования для упражнения.
        [] Название упражнения (Push-ups).
        [] Секция с предыдущими результатами и текущими повторениями и весом.
        [] Отметка выполнения упражнения.
    [] Profile (Профиль):
        [] Аватар пользователя.
        [] Имя пользователя (Max HTMLLESSONS).
        [] Количество минут тренировок.
        [] Количество выполненных тренировок.
        [] Суммарный вес поднятого груза.
        [] Фотографии "до" и "после".
    [] Create workout (Создание тренировки):
        [] Изображение фона.
        [] Поле для ввода названия тренировки.
        [] Кнопка для добавления упражнений.
        [] Кнопка "Create" (Создать).
    [] Create exercise (Создание упражнения):
        [] Изображение оборудования для упражнения.
        [] Поле для ввода названия упражнения.
        [] Поле для ввода времени.
        [] Иконки типов упражнений.
        [] Кнопка "Create" (Создать).

Frontend:
    [] React
    [] Axios
    [] TanStack Query
    [] SCSS

Endpoints:
    [] Home (Главная):
        [] GET /home: Получение основной информации для главной страницы (текущая тренировка, прогресс, рекомендации).
    [] Open Menu (Открытое меню):
        [] GET /menu: Получение пунктов меню для текущего пользователя.
    [] Workouts (Тренировки):
        [] GET /workouts: Получение списка всех тренировок.
        [] POST /workouts: Создание новой тренировки.
        [] GET /workouts/{id}: Получение информации о конкретной тренировке.
        [] PUT /workouts/{id}: Обновление информации о конкретной тренировке.
        [] DELETE /workouts/{id}: Удаление тренировки.
    [] Exercise (Упражнение):
        [] GET /exercises: Получение списка всех упражнений.
        [] POST /exercises: Создание нового упражнения.
        [] GET /exercises/{id}: Получение информации о конкретном упражнении.
        [] PUT /exercises/{id}: Обновление информации о конкретном упражнении.
        [] DELETE /exercises/{id}: Удаление упражнения.
    [] Profile (Профиль):
        [] GET /profile: Получение информации о профиле текущего пользователя.
        [] PUT /profile: Обновление информации профиля.
    [] Create workout (Создание тренировки):
        [] POST /workouts: Создание новой тренировки (используется также в разделе тренировок).
    [] Create exercise (Создание упражнения):
        [] POST /exercises: Создание нового упражнения (используется также в разделе упражнений).
    [] Authentication (Аутентификация):
        [] POST /login: Авторизация пользователя.
        [] POST /logout: Выход пользователя из системы.
        [] POST /register: Регистрация нового пользователя.

Backend:
    [] Express.js
    [] PostgreSQL
    [] Prisma
    [] JWT
